# Network Site Stats Plugin

## Mô tả
Plugin quản lý và thống kê các site con trong mạng lưới WordPress Multisite.

## Tính năng
- Hiển thị danh sách tất cả site con
- Thống kê số bài viết của từng site
- Hiển thị ngày đăng bài mới nhất
- Giao diện tích hợp với WordPress Admin

## Cài đặt
1. Upload thư mục `network-site-stats` vào `/wp-content/plugins/`
2. Vào Network Admin > Plugins
3. Chọn "Network Activate" để kích hoạt cho toàn mạng lưới

## Sử dụng
- Truy cập Network Admin > Site Stats để xem thống kê

## Yêu cầu
- WordPress Multisite đã được kích hoạt
- Quyền Super Admin
